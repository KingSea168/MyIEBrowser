开发工具: Microsoft Visual Studio 2005

值与引用类型的传递, C#网络应用, 其他基本技能点.

ST122_周立洋: 窗体UI的实现, 部分技术实现, 程序调试

使用手册:

 

快捷键:

Alt + F1: 打开帮助文档

Alt+F11: 打开/关闭全屏显示

Ctrl+左: 上一页

Ctrl+右: 下一页.

新标签页:

可以创建一个新的空白标签页.

输入网址:

当有输入记录时, 如果有你输入的字符完全匹配的记录地, 系统会自动补全网址, 实现快速浏览, 点击跳转后, 会跳转到相应的页面.

Google搜索:

在搜索栏中输入搜索内容后, 点击搜索按钮, 即可显示谷歌搜索引擎的搜索内容.

关闭当前标签页:

点击关闭当前标签页按钮后, 会关闭当前选中的页面.

关闭全部标签页:

点击关闭全部标签页按钮后, 全关闭全部的打开页面.

网页浏览:

当输入网址或通过其他方式打开一个页面后, 在页面加载时, 状态条中的进度条会提示当前的载入进程, 当载入完毕后, 会在主窗体中显示该页面的内容, 此时您可以对页面进行预览,图片的下载, 文本的复制,  源文件的查看或者打印等操作.

打开本地文件:

文件 C 打开 C 选择文件 C 确定 C 可以能过畅游网络浏览器浏览你喜欢的文件了.

保存网页:

已经打开一个网页 C 文件 C 保存网页 C 选择保存路径与文件名 C 确定 C 保存成功了

打印文件内容:

已经打开一个网页或其他文件 C 文件 C 打印 C 打印 C 选择打印机 C 打印成功

也可以先预览打印效果与设置打印机

查看文件属性:

已经打开一个网页或其他文件 C 文件 C 属性 C 可以查看当前文件的属性了

自定义窗体UI:

查看 C 视窗 C 选择要显示或隐藏的控件后, 即可实现效果, 或者在工具栏或菜单栏上单击鼠标右键, 也可以在工具 C 首选项 C 窗体显示中, 保存你的自定义效果.

前进, 后退, 停止, 刷新, 第一页, 最后一页:

在浏览网页的同时, 系统会自动判断当前页是否有前进(最后一页)或后退(第一页), 在当前页加载时, 点击停止会停止加载, 在当前页上点击刷新可以重新加载当前网页.

以上功能也可以通过菜单栏 C 查看 或者 在网页中右键, 完成操作.

主页:

当点击主页按钮时, 页面会跳转到主页中, 主页可以通过首选项自定义.

搜索页:

当点击 查看 C 搜索项后, 页面跳转到搜索页中, 搜索页可以通过首选项自定义.

查看源文件:

已经打开一个网页文件(HTML, JSP, XML, ASPX格式) C 查看 C 查看源文件 C 打开查看源文件窗体.  在源文件窗体中可以通过点击保存按钮, 实现源文件的保存.

全屏:

查看 C 全屏 C 页面会全屏浏览, 点击退出全屏按钮会退出全屏浏览状态.

历史(只限保存10条历史记录):

当浏览了一个网页或文件后, 系统会自动记录最近浏览的10条历史记录, 可以通过 历史 C 选择相应的历史记录, 再次打开浏览过的页面. 当退出程序后, 程序会继续保留历史记录, 再次打开程序, 依然会存在历史记录. 历史记录可以通过首选项 C 删除记录清除.

书签(只限自定义10条书签):

书签 C 添加书签 C 可以快速添加一条书签, 通过书签管理可以添加, 删除, 修改书签内容,

当程序退出后, 程序会继续保留书签内容, 再次打开程序, 自定义的书签依然会存在.

新闻订阅:

工具 C 新闻订阅 C 窗体左侧会出现一个新闻阅读窗体, 可以通过点击工具栏第一个按钮实现新闻阅读窗体的显示与隐藏, 点击添加频道, 输入地址与标题后, 会自动下载相应的新闻内容并显示, 点击新闻标题后, 就可以通过浏览窗口查看最新的新闻内容了. 也可以对新闻频道进行修改, 删除, 更新等操作. 点击关闭后, 系统会自动保存您订阅的新闻内容, 下次打开程序, 依然可以继续浏览.

邮件收发:

工具 C 邮件收发 C 打开邮件收发窗体 C 邮件设置 C 输入您的邮件地址与邮箱配置后 C 点击获取邮件 C 即可查看您的邮箱内容, 点击发送邮件, 即可用该邮件地址与用户名发送一封邮件了. 点击后台运行, 窗体进程不会被终止, 会在任务栏中显示一个任务图标, 点击任务图标, 会再次显示邮件窗体. 邮件配置可以保存.

首选项:

工具 C 首选项 C 启动页, 可以设置打开程序时, 自动加载的网页, 有空标签页, 主页, 自定义标签页可选择. 主页 C 参见主页, 搜索页 C 参见搜索面. 窗体显示 C 参见视图. 删除记录 C 参见历史. 恢复缺省设置 C 可以一键还原为系统最初的设置. 当设置完成后, 程序会自动保存配置信息, 再次打开程序后, 配置内容即可生效.

程序更新(未完成实现):

帮助 C 程序更新 - 可以查看当前的文件版本号与更新日期, 未能完成更新功能.

检查连接:

帮助 C 检查连接 C 可以快速检查当前网络是否连接.
